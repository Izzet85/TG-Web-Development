type selector voorbeeld: body{ background-color: blue; } Dit is een css selector waabij de styling word toegepast op basis van de geselecteerde html element naam ook wel node name genoemd.Het voordeel hiervan is dat je met een selectie heel veel dingen aan element kunt veranderen. In het voorbeeld word op de body element een blauwe achtergrondkleur toegeapst. descendant selectors voorbeeld: div p { background-color: yellow; } Bij descendant selectors worden alle afstammelingen van een bepaalde element geselecteerd. Het voordeel hiervan is dat je specifieke html elementen kunt voorzien van unieke styling. In het voorbeeld boven is de p een afstammeling van de div, omdat de p binnen de div element is genesteld. Op deze manier kun je afstammelingen van bepaalde html elementen selecteren. class selectors voorbeeld: .intro { background-color: yellow; } Bij class selector selecteer je die elementen die voorzien zijn van een class attribute.Het voordeel hiervan is dat je de styling een keer hoeft op te stellen en dat je hem daarna op elke gewenste html element kunt toepassen. In het voorbeeld staat een intro atribute met een background-color property met een waarde van yellow. Elke element die is voorzien van een intro atribute word voorzien van een gele background-color.
