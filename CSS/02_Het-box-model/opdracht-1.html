Wat zijn de eigenschappen van block boxes en inline boxes? Een block element begint altijd op een nieuwe lijn en neemt in volle breedte de ruimte in beslag. een block element heeft vergeleken met een inline element wel een top en en bottom margin. Een inline element start niet op een nieuwe regel en neemt niet het gehele breedte in beslag, maar zo breed als nodig.
