1.static positioning dit is default position die html elementen hebben.Elementen die een static position hebben kunnen net beinvloed worden door top,bottom,left en right properties. 2.relative positioning Een element met een position property ingesteld op relative kan zijn positie relatief tot zijn normale positie veranderen. De positie van het element kan veranderd worden op basis van de properties top,bottom,left en right. 3.absolute positioning Een element met de position property ingesteld op abolute is gepositioneerd op basis van zijn ouders die position relative hebben. Als de element met een absolute position geen ouders heeft word de positie bepaald door de body van het html document. 4.fixed positioning Een elemenet met een property position ingesteld op fixed is gepositioneerd relatief tot de viewpoort,wat betekent dat de element altijd vast staat op hetzelfde plek ook al scrroll je naar beneden de pagina. 5.sticky positioning Een element met een position property ingesteld op sticky is gepositioneerd op basis van de scroll positie van de gebruiker. De element is relatie gepositioneerd totdat het de offset position bereikt in de viewport.
